### What is LumineDroid?

**LumineDroid** is a clean, lightweight, and performance-focused custom Android OS based on AOSP. Built with minimalism and efficiency in mind, it offers a bloat-free experience, deep customization, and modern aestheticsâ€”ideal for those who want complete control over their device.

### Source Installation

```
repo init -u https://github.com/LumineDroid/platform_manifest.git -b v --git-lfs
repo sync -c -j$(nproc --all) --force-sync --no-clone-bundle --no-tags
```
